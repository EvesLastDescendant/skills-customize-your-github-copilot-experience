
# 🎮 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman game in Python to practice string manipulation, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Set Up the Game

#### Description
Create a game that picks a hidden word, displays blanks for each letter, and lets the player guess letters one at a time.

#### Requirements
Completed program should:

- Choose a word from a predefined list
- Display the word as underscores (for example, `_ _ _ _`)
- Accept single-letter guesses from the user
- Reveal correctly guessed letters in the correct positions

### 🛠️ Track Progress and End the Game

#### Description
Add gameplay logic to track incorrect guesses, limit attempts, and end the game when the player wins or loses.

#### Requirements
Completed program should:

- Track the number of incorrect guesses remaining
- Show the letters the player has already guessed
- End the game when the word is guessed or attempts are exhausted
- Display a win or loss message at the end
