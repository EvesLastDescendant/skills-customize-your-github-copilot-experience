# 🎮 Assignment: Text Adventure Game

## 🎯 Objective

Create a simple text-based adventure game in Python where the player makes choices, explores locations, and tracks progress through the story.

## 📝 Tasks

### 🛠️ Build the Story Structure

#### Description
Design a small game world with at least two locations and a few choices for the player to make.

#### Requirements
Completed program should:

- Present the player with an opening scene
- Give the player at least two choices to make
- Use conditional logic to move between different story paths
- Show a clear ending for the chosen path

### 🛠️ Track Player State

#### Description
Add simple state tracking so the game can remember important information during play.

#### Requirements
Completed program should:

- Store at least one piece of player information such as health, score, or collected items
- Update that information based on the player’s choices
- Display the current state during the game
- Use the state to affect the final outcome

### 🛠️ Add a Replay Option

#### Description
Make the game more interactive by allowing the player to play again after finishing.

#### Requirements
Completed program should:

- Ask the player whether they want to play again
- Reset the game state for a new round
- Keep the game running smoothly across multiple plays
